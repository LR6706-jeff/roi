# 📊 电商投流 ROI 计算器

> 实时核算 · 智能 AI 诊断 · 降本增效  
> 微信小程序 | 支持商家版 & 达人版

---

## 功能介绍

### 🏪 商家版
| 输入项 | 说明 |
|--------|------|
| 商品售价 | 单件售出价格（元） |
| 进货成本 | 单件进货价（元） |
| 物流包装费 | 单件快递+包材费（元） |
| 平台扣点 | 平台佣金比例（%） |
| 退货率 | 预估退货百分比（%） |
| 广告花费 | 本期投流总花费（元） |
| 成交订单数 | 实际出单量（单） |

**输出结果：**
- 保本底线 ROI（低于此值即亏损）
- 实际综合 ROI
- 预估净利润
- 综合利润率

---

### 🎬 达人版
| 输入项 | 说明 |
|--------|------|
| 商品售价 | 单件售价（元） |
| 佣金比例 | 平台/商家给的佣金率（%） |
| 预估出单量 | 预计带货单量（单） |
| 广告花费 | 投流总花费（元） |
| 提现费率 *(可选)* | 平台提现手续费（%） |
| 1h退款率 *(可选)* | 1小时内退款率，此类订单不计佣金（%） |

**输出结果：**
- 保本底线 ROI（含提现费率与1h退款率修正）
- 实际综合 ROI（广告费为0时显示 ∞）
- 预估佣金净收益
- 投流分成利润率

---

### ✨ AI 智能诊断
- 输入数据后一键触发
- 优先调用微信云开发云函数（接入大模型 API）
- 云端不可用时自动降级为本地规则诊断引擎
- 诊断结果自动保存至历史记录

## ROI 公式说明

```
实际 ROI     = 销售额(GMV) / 广告花费
保本 ROI     = 售价 / 单品利润（商家版）
达人保本 ROI = 1 / (佣金率 × (1 - 1h退款率) × (1 - 提现费率))
```

> 实际 ROI > 保本 ROI → ✅ 盈利  
> 实际 ROI < 保本 ROI → ❌ 亏损

## 部署指南

### 1. 配置 AppID

在 `project.config.json` 中替换为你的真实 AppID：
```json
{
  "appid": "你的AppID"
}
```

### 2. 开通云开发环境

1. 打开微信开发者工具，点击顶部「云开发」按钮
2. 创建环境（建议命名 `roi-prod`），选择「按量计费」
3. 记录环境 ID（格式：`roi-prod-xxxxxx`）

### 3. 更新云环境 ID

在 `app.js` 中填入你的环境 ID：
```js
wx.cloud.init({ env: '你的环境ID' });
```

### 4. 部署云函数

在开发者工具左侧目录树中：
1. 展开 `cloudfunctions/aiDiagnostic`
2. 右键 → **「上传并部署：云端安装依赖」**
3. 等待弹窗提示「上传成功」

### 5. 配置大模型 API Key

在 `cloudfunctions/aiDiagnostic/index.js` 中填入你的 API Key：
```js
const API_KEY = '你的Key';
```
